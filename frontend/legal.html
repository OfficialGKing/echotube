Terms of Service

Last updated: December 29, 2024

1. Services Overview

EchoTube provides tools for YouTube content creators to manage their channel content, including:

  • Comment management and interaction
  • Video analytics and monetization tracking
  • Channel performance metrics
  • Demo mode for API quota management

2. YouTube API Services

This application uses YouTube API Services and requires you to grant specific permissions to:

  • Access your YouTube account data
  • View and manage your videos
  • View and respond to comments
  • Access monetization and analytics data

3. Data Collection and Usage

We collect and process the following data:

  • Video metadata (titles, descriptions, statistics)
  • Comment data and interaction history
  • Monetization status and earnings estimates
  • Channel performance metrics

4. API Quota Management

To ensure service availability:

  • We implement fair usage limits on API requests
  • Demo mode is provided when API quota is exceeded
  • Cached data may be used to reduce API consumption

Privacy Policy

Last updated: December 29, 2024

1. Data Collection

We collect the following information through the YouTube API:

  • Channel information
  • Video data including views, likes, and monetization status
  • Comment data and interaction history
  • Estimated earnings data (for monetized channels)

2. Data Usage

Your data is used to:

  • Display channel analytics and performance metrics
  • Calculate estimated earnings
  • Provide comment management features
  • Track video performance and monetization status
  • Support demo mode functionality

3. Data Storage

We do not permanently store your YouTube data. All data is fetched in real-time from YouTube's APIs and cached temporarily for performance.

GDPR Compliance

Last updated: December 29, 2024

1. Data Subject Rights

Under GDPR, you have the right to:

  • Access your personal data
  • Rectify inaccurate personal data
  • Erase your personal data
  • Restrict processing of your data
  • Data portability
  • Object to processing of your data

2. Data Processing

We process the following data categories:

  • YouTube account information
  • Channel performance data
  • Video analytics and monetization data
  • Comment interaction history
  • Temporary cache data for API quota management

3. Legal Basis

We process your data based on:

  • Your consent when connecting your YouTube account
  • Legitimate interests in providing channel management services
  • Contractual necessity to provide our services

4. Data Retention

We retain your data only for as long as necessary to provide our services. You can revoke access to your YouTube account at any time through your Google Account settings.

For questions about our legal policies, please contact us at legal@echotube.com
